Building "Todo Intelligence" (started 2026-07-02): an offline-first always-on-top sidekick desktop app. Paste raw workplace messages → local LLM (Ollama qwen2.5:3b) extracts structured tasks (title, subtasks, deadline hard/soft, priority, clarifying questions). Features: async enrichment UX (~5s LLM latency), open-loop clarification Q&A, morning briefing on first open of day, Today/Week/Later/Done views, element legend, tray + global hotkey capture. Windows first, macOS later.

**Key decisions:** Electron + electron-vite + React + TS (Tauri rejected — no Rust/MSVC on machine). vite pinned to ^7 (electron-vite@5 incompatible with vite 8). Atomic JSON file store (no native modules). Per-user NSIS installer (no admin). LLM is assistive, never a gatekeeper — manual editing always works; app degrades gracefully when Ollama is down.

Design artifacts: docs/DESIGN.md, docs/ARCHITECTURE.md, docs/LLM_PIPELINE.md (produced by design workflow). Related: [[user-environment]]
