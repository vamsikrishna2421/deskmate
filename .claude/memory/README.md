# Claude Code session memory (portable copy)

These files are the persistent memory from the DeskMate development sessions —
project state, decisions, and working preferences. To continue development with
full context on another device, either:

1. Copy these files into that machine's
   `~/.claude/projects/<this-project's-slug>/memory/` directory, or
2. Just ask Claude to read `.claude/memory/` at the start of the session.

Keep this in sync when major decisions land (source of truth is the local
memory dir on the primary machine).
